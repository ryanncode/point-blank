Welcome to Point Blank Documentation

This website provides in-depth information about the Point Blank VS Code extension, including its core features, architectural design, and how to contribute to the project.

Get the extension on the VS Code Marketplace!

What's New in 6.1.0

Point Blank now includes a powerful new inline query language! You can now dynamically pull information from across your workspace directly into your notes. Use `LIST` to create lists of links to files or content blocks, or use the new `TRANSCLUDE` keyword to embed content directly. Learn more about this powerful new feature on the Queries page.
