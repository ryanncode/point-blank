Welcome to Point Blank Documentation

This website provides in-depth information about the Point Blank VS Code extension, including its core features, architectural design, and how to contribute to the project.

Get the extension on the VS Code Marketplace!

What's New in 0.6.0

Point Blank has undergone a significant refactor of its template system! We've introduced new syntax for defining file-based templates and for inline node insertion. Learn more about these powerful new features on the Features page.

Features

Detailed explanations of Point Blank's intelligent bullet point styling, hierarchical outlining, typed nodes, template expansion, and focus mode.

Queries

Learn how to use Point Blank's powerful inline query language to search for notes and data across your workspace, with support for filtering, sorting, and different scopes.

Architecture

A deep dive into the modular, performant, and maintainable architecture of Point Blank, centered around an immutable document model.

Contributing

Information on how to get involved with the Point Blank project, including development setup and guidelines.
